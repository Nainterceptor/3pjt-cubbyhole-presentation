Company

Cubbyhole is a recent startup which has raised 124 000 euros with help of "The Family".
The aim for the eponymous product is to innovate in the cloud domain with the use of mongoDB to store files.

Competitions

  • Dropbox
  • OVH
  • OwnCloud

Targets

Mainly B2B product

  • Companies with HA constraints
  • Companies with bandwidth constraints
  • Companies with multilocalization constraints (CDN)

Technologies choices

  • API and Web application : NodeJS
  • Storage and database : MongoDB
  • Desktop application : Qt
  • Mobile application : Android

Different ways to store files

  • Files splitted into 16MB documents
  • Documents are duplicated into replica set
  • Documents are mounted in memory after reading
